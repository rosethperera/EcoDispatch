## 🚨 Troubleshooting

### Portfolio Won't Show
**Check:**
1. Folder structure matches guide exactly
2. Markdown formatting is correct
3. Image links use proper path
4. GitHub pages is enabled

### Video Has Sync Issues
**Fix:**
1. Re-export from DaVinci
2. Use FFmpeg to fix timing
3. Re-record audio if too far off
4. Use guide's sync instructions

### Streamlit App Won't Deploy
**Check:**
1. requirements.txt has all dependencies
2. No local imports of non-standard libraries
3. Check Streamlit logs for errors
4. Verify repo is public

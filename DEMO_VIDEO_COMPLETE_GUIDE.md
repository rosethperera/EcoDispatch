# Demo Video Guide

Record the project as a prototype simulator, not as a fixed-results product pitch.

## Suggested 60-Second Script

"EcoDispatch is a simulation prototype for carbon-aware data center energy optimization. It models how a site could coordinate grid power, on-site solar, battery storage, and flexible workloads. The dashboard compares five dispatch strategies, shows cost and emissions tradeoffs, and makes load-served accounting explicit so no strategy gets a fake win. Scenario presets make it easier to see how different operating conditions change the outcome."

## Show These Screens

1. Strategy comparison table
2. Strategy explanation section
3. Scenario profile controls
4. Dispatch timeline
5. Battery state-of-charge chart

## Avoid Saying

- fixed percentage savings unless tied to one clearly labeled run
- "production-ready"
- "live deployed" unless verified
- outdated test or coverage counts

## 🎥 Recording Instructions

### Option A: Use OBS Studio (Free, Professional) ⭐ RECOMMENDED

**Setup (5 minutes):**
1. Download OBS Studio: https://obsproject.com/
2. Open OBS
3. Click "+" under Sources → Display Capture (or Window Capture)
4. Select your screen/window
5. Set resolution to 1920x1080 @ 60fps in Settings
6. Hit "Start Recording"

**Recording Steps:**
1. Open your dashboard: `streamlit run dashboard.py`
2. Zoom to 125% or 150% for readability
3. Record at normal speed
4. Speak clearly and slowly
5. Do multiple takes if needed (pick the best one)

**Recording Checklist:**
- [ ] Dashboard is clean (no errors visible)
- [ ] Text is readable (zoom in if needed)
- [ ] Speak clearly and at good pace
- [ ] Show different strategies
- [ ] Interact with parameters
- [ ] Let charts render fully before moving on

### Option B: ScreenFlow (Mac) or ShareX (Windows)

**For Mac:**
1. Launch ScreenFlow
2. Select region to record
3. Click Record
4. Do your demo
5. Click Stop
6. Edit and export

**For Windows:**
1. Open ShareX
2. Click "Start Recording" (Shift+Print Screen)
3. Do your demo
4. Press Ctrl+C to stop
5. Export video

### Option C: Built-in Tools

**Windows 11:**
1. Press Win+G (Game Bar)
2. Click "Start recording"
3. Do your demo
4. Press Win+Alt+R to stop

**Mac:**
1. Press Cmd+Shift+5
2. Select "Record"
3. Do your demo
4. Hit stop

## 🎙️ Audio Recording (Separate from Video)

### Best Option: Use Audacity (Free)
1. Download: https://www.audacityteam.org/
2. Click Record (red button)
3. Read the script slowly and clearly
4. Click Stop
5. Export as MP3: File → Export → Export as MP3

### Recording Tips:
- ✓ Use a quiet room
- ✓ Speak 20% slower than normal
- ✓ Pause between sentences
- ✓ Don't rush
- ✓ Smile while speaking (you'll hear it!)
- ✓ Record each scene separately, then combine

### Script Timing Guide:
```
Scene 1 (Intro): 8 seconds
  - Read at natural pace
  - Pause 2 seconds at end

Scene 2 (Problem): 8 seconds
  - Speak clearly
  - Emphasize "400 gCO2"

Scene 3 (Solution): 14 seconds
  - Build excitement
  - Pace: moderate to faster

Scene 4 (Results): 20 seconds (+caption time)
  - This is the CLIMAX
  - Emphasize "27%", "21%", "40%"
  - Let each stat sink in

Scene 5 (Capabilities): 8 seconds
  - Quick, energetic

Scene 6 (CTA): 2 seconds
  - "Try it now!"
```

## ✂️ Video Editing

### Option 1: DaVinci Resolve (Free, Professional) ⭐ BEST

**Step-by-Step Editing:**
1. Download: https://www.blackmagicdesign.com/products/davinciresolve/
2. New Project → Import your screen recording
3. Add audio track → Import your voiceover
4. Sync timing:
   - Video should play at normal speed
   - Audio should match scenes
   - Add 0.5 second fade out at end

**Adding Text Overlays:**
1. Fusion tab
2. Click "Text+" to add title
3. Type your text
4. Adjust timing to match scenes
5. Color: green (#2ecc71) for your brand

**Adding Transitions:**
- Between scenes: 0.5-1 second fade
- On stat reveals: quick cut (no fade)

**Exporting:**
1. Deliver tab
2. Format: MP4
3. Codec: H.264
4. 1920x1080, 30fps
5. Export

### Option 2: CapCut (Free, Quick) 

1. Download CapCut
2. Import video
3. Click audio → add your voiceover
4. Click text → add scene titles
5. Add transitions between scenes
6. Export

### Option 3: Adobe Premiere Pro (Professional)

If you already have it, same process as DaVinci Resolve.

## 🎨 Text Overlay Specs

Use these for consistency:

**Title/Headlines:**
- Font: Arial or Helvetica (clean, professional)
- Size: 48-60pt
- Color: #2ecc71 (your brand green)
- Position: Center or upper-third
- Duration: 3-4 seconds on screen

**Statistics:**
- Font: Arial Bold
- Size: 36pt
- Color: White on dark (or green on white)
- Position: Lower-third
- Duration: 3-5 seconds each

**Captions:**
- Font: Arial
- Size: 24pt
- Color: White
- Position: Bottom
- Duration: Until you say the next phrase

## 📥 Combining Video + Audio

### Using DaVinci Resolve:
1. Drag video to timeline
2. Right-click → "Detach Audio" (if it has audio)
3. Drag voiceover MP3 to audio track below
4. Align timing:
   - Start of video = start of audio
   - Adjust video clips to match speech
5. Add transitions between video clips
6. Export with both tracks

### Using CapCut:
1. Import video
2. Tap Audio at bottom
3. Tap "Add Audio" → select your MP3
4. Drag to align timing
5. Tap Export

### Using Adobe Premiere:
1. Import video to V1 track
2. Import audio to A1 track
3. Align clips in timeline
4. Export as MP4

**Pro Tip:** Video should be 100% synced with audio. Test by:
- Playing video at 0.75x speed
- Listening for sync issues
- Adjusting video clips if needed

## 🎬 Recording Tips for Best Quality

### Video Quality:
- Record at 1920x1080 minimum (4K if possible)
- 60fps for smooth movement
- Good monitor brightness (dashboard should be visible)
- Zoom dashboard to 125% for text readability

### Audio Quality:
- Use a decent USB microphone (~$30)
- Record in quiet room (no fans, AC noise)
- Speak clearly at natural pace
- Leave 1-2 second pauses between scenes
- Re-record sections with mistakes

### Dashboard Appearance:
- Light theme (your white dashboard is perfect)
- Run on clean system (no system notifications)
- Maximize window to fill screen
- Disable taskbar/dock if possible
- Test all links work before recording

## 🔧 Compressing Your Video

**If file size is too large:**

### Using FFmpeg (Free):
```bash
# High quality, smaller file
ffmpeg -i demo.mp4 -c:v libx264 -crf 23 -c:a aac -b:a 128k demo_compressed.mp4

# For web (even smaller)
ffmpeg -i demo.mp4 -c:v libx264 -preset slow -crf 28 -c:a aac -b:a 96k demo_web.mp4
```

### Using Online Tools:
- Handbrake (GUI version)
- CloudConvert.com
- Kapwing.com

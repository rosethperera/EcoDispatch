# Screenshots

This folder stores the visuals shown in the GitHub README.

Recommended files:

- `dashboard-overview.svg` or `dashboard-overview.png`
- `energy-dispatch-chart.png`
- `battery-soc-chart.png`
- `strategy-comparison.png`

Suggested approach:

1. Run `python demo.py` to generate charts in `outputs/`.
2. Capture the Streamlit dashboard after running `streamlit run dashboard.py`.
3. Replace the current representative visuals with stronger screenshots as needed.

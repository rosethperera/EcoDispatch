# Portfolio Package Summary

This package should present EcoDispatch as an engineering prototype, not as a fully validated commercial system.

## Best Current Description

EcoDispatch is a Python simulation prototype for carbon-aware data center energy optimization. It models dispatch across grid, solar, battery storage, and flexible workloads, and provides a Streamlit dashboard for scenario analysis and strategy comparison.

## Portfolio Highlights You Can Defend

- Five dispatch strategies compared on a shared demand basis.
- Streamlit dashboard with scenario controls and strategy explanations.
- Battery and solar models with configurable system parameters.
- Optional real weather and optional Electricity Maps carbon/price inputs.
- Prototype hardware integration paths through Arduino and Raspberry Pi code.
- Unit-tested simulation and data-integration behavior.

## Portfolio Highlights You Should Remove

- Fixed savings claims like 27%, 21%, or 40%.
- Fixed workload-shift numbers like 1,800 kWh.
- Coverage claims like 98% or 42 tests.
- "Production-ready."
- National-scale emissions savings projections.

## Recommended Website Framing

- Category: Sustainable infrastructure
- Positioning: Simulation and scenario-analysis prototype
- Tone: Transparent, technical, and evidence-based

## Suggested Short Blurb

"EcoDispatch is a carbon-aware energy-dispatch simulation prototype for data centers. It compares multiple strategies for using grid power, solar, batteries, and flexible workloads under different price and carbon conditions."

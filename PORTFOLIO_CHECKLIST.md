# Portfolio Checklist

## Keep

- Prototype framing
- Strategy comparison
- Scenario presets
- Dashboard screenshots
- Battery and solar model discussion
- Optional real data inputs
- Hardware integration as a prototype path

## Remove

- Unsupported fixed percentages
- Unsupported workload-shift totals
- Unsupported coverage/test-count claims
- "Production-ready"
- "Live deployment" unless separately verified
- National-scale impact projections

## Safe One-Liner

"I built a simulation prototype for carbon-aware data center energy dispatch that compares multiple strategies for using grid power, solar, batteries, and flexible workloads."

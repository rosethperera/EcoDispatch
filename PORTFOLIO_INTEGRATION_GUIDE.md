# Portfolio Integration Guide

Use this wording when integrating EcoDispatch into a portfolio site.

## Safe Description

"EcoDispatch is a Python simulation prototype for carbon-aware data center energy optimization. It models dispatch across grid, solar, battery storage, and flexible workloads, and includes a Streamlit dashboard for scenario analysis and strategy comparison."

## Safe Tags

- Python
- Streamlit
- SciPy
- Energy systems
- Sustainable infrastructure
- Simulation

## Safe Highlights

- Five dispatch strategies
- Scenario presets for clearer comparisons
- Battery and solar models
- Flexible workload shifting experiments
- Optional external data inputs
- Prototype hardware integration concepts

## Claims To Exclude

- Fixed emissions or cost savings percentages
- Coverage claims not backed by tooling in the repo
- Production-readiness
- Verified deployment claims unless you separately confirm them
